Excel: Minimizing data entry errors with basic data validation

Data Validation allows you to control what kinds of data can be entered into a cell by creating “rules.” This can help reduce data entry errors.


Hold Ctrl (for Windows) / Cmd (for Macs) to select non-adjacent cells.

Excel has a number of options when applying data validation. These include:

  • Text Length: This determines how many characters are allowed in the cell. You may select an exact number or a range.
  • Date: This requires the cell to be a valid date. You will need to choose the allowed date range.
  • List: Users will have a drop down list they can select from. You will need to select an existing range to reference to.

The title and message will be seen when the user selects the cell.

This is a message alert that will show if the user's input does not match the validation rules. You may choose between 3 different errors depending on how strict you want your validation to be.

  • Stop: Requires users to retry or cancel. It will not allow the user to leave the cell until it is valid.
  • Warning: Informs the user of the error. They can opt to continue or return to the cell from the alert.
  • Information: Informs the user of the error. The alert has one button to continue onwards.
